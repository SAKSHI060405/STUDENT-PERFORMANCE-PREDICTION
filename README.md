# Student Performance Prediction System

A beginner Machine Learning project built using Python and Scikit-learn to predict student academic performance based on study-related factors.

## Features

* Student score prediction
* Data preprocessing
* Linear Regression model
* Data visualization using Matplotlib

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## Output

The model predicts final student performance using factors such as:

* Study Hours
* Attendance
* Previous Scores
